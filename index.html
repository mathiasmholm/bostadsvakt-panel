🏡 Bostadsvakt – kontrollpanel

Panelen pratar direkt med ditt privata GitHub-repo. Klistra in en personlig åtkomsttoken – den sparas bara i din webbläsare.

Hur skapar jag en token?
  1. GitHub → din profilbild → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
  2. Repository access: Only select repositories → välj bostadsvakt
  3. Permissions → Repository permissions: Contents: Read and write samt Actions: Read and write
  4. Generera, kopiera och klistra in här
