Bostadsvakt

Logga in med en personlig åtkomsttoken. Den sparas enbart i din webbläsare och skickas aldrig någon annanstans än till GitHubs API.

Hur skapar jag en token?
  1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
  2. Repository access: Only select repositories → bostadsvakt
  3. Permissions: Contents och Actions, båda Read and write
  4. Generera, kopiera och klistra in här
